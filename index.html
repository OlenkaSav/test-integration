How to work with Contacts in Hubspot API

How to work with Contacts in Hubspot API Contacts represent records about people in Hubspot API. They are typically associated with Companies and Activities such as Emails, Meetings, Notes, etc.

Contact Fields

Field Name API Key Type Description
First Name first_name string First name of the person
Last Name last_name string Last name of the person
Address address object Address associated with the person
Full address.full string Full address associated with the person
Street address.streeet string Street address
